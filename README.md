# HTML Only Page Recreation

### Repository for DUMP Internship: Week 8 - HTML + CSS

## Task

This project represents a **static HTML page recreation** based on the following reference website:  
https://html-only.netlify.app/

The goal of the assignment is to **recreate the layout, structure, and content** of the provided page using **only HTML**.

The page should visually and structurally resemble the original, while being implemented as a **single HTML file**.

## Setup

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
